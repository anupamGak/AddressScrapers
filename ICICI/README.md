ICICI Bank
==========

Introduction
------------

Scraps address of branches and ATMs from the webpage [ICICI Branch Locator](http://maps.icicibank.com/mobile/).

Dependencies
------------

The scrawler depends on ``requests``.

Usage
-----

For branches:

```sh
$ python icici-branch_spider.py
```

For ATMs:

```sh
$ python icici-atm_spider.py
```
